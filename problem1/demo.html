Problem 1: Three Ways to Sum to n

Problem Statement

Input: n - any integer

Output: Sum of integers from 1 to n

Example: sum_to_n(5) = 1 + 2 + 3 + 4 + 5 = 15

Approach A: Iterative Loop

var sum_to_n_a = function(n) { let sum = 0; for (let i = 1; i <= n; i++) { sum += i; } return sum; };

Approach B: Recursive

var sum_to_n_b = function(n) { if (n <= 0) { return 0; } return n + sum_to_n_b(n - 1); };

Approach C: Mathematical Formula

var sum_to_n_c = function(n) { return n * (n + 1) / 2; };
